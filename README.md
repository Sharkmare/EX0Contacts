# EX0Contacts

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) that gives you full control over contact list sorting order.

Vanilla Resonite hardcodes contact ordering. This mod makes every status priority configurable.

## What it does

- **Configurable sort priority** for every online status: Online, Sociable, Away, Busy, Offline, Headless, Incoming Requests
- **Lower number = higher in list** - set any status to 0 to pin it to the top
- All settings accessible via the in-game mod settings panel

## Default sort order

| Priority | Status | Vanilla |
|----------|--------|---------|
| 0 | Online | 1 |
| 0 | Sociable | 0 |
| 2 | Incoming Requests | 2 |
| 3 | Away | 3 |
| 4 | Busy | 4 |
| 5 | Headless | ? |
| 6 | Offline/Invisible | 5 |


## Installation

1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader)
2. Place `EX0Contacts.dll` into your `rml_mods` folder

Settings appear under **EX0Contacts** in the mod settings panel.
